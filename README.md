# InstruCal Pro Website

Site vitrine professionnel pour l'application InstruCal Pro.

## Publication GitHub Pages
1. Créer un dépôt GitHub : `instrucal-pro-website`
2. Uploader tous les fichiers
3. Aller dans Settings > Pages
4. Source : Deploy from a branch
5. Branch : main
6. Folder : /root
7. Save

Lien attendu :
https://Tattaztech.github.io/instrucal-pro-website/

## Images
Mettre les captures dans `images/` puis remplacer les blocs placeholder dans `index.html`.
